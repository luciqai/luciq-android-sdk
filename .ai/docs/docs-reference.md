# Luciq SDK — Documentation Reference

## Authoritative Source

The official, up-to-date documentation for the Luciq SDK lives at **docs.luciq.ai**. When the local steering files don't cover a topic in enough detail, fetch the relevant page from the documentation site.

## When to Query docs.luciq.ai

Use your web fetch capability to retrieve content from docs.luciq.ai when the user asks about:
- Specific product behavior or configuration details not covered in local files
- Performance specifications (SDK launch time, memory/CPU overhead, size impact)
- Pricing, plan limitations, or feature availability details
- Advanced configuration options
- Platform-specific nuances (Flutter, React Native cross-platform behavior)
- Changelog or version-specific changes
- Any topic where the local steering files provide insufficient detail

1. **First**, answer from local steering files (setup, features, integration, troubleshooting)
